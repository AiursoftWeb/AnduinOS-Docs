# Touchpad Gestures

AnduinOS uses the Wayland display server by default, which brings a buttery-smooth, 1:1 multi-touch gesture experience that rivals macOS. If you are using a laptop, mastering these gestures will fundamentally change how you interact with the system.

## Navigation and Workspaces

Multi-tasking on AnduinOS relies heavily on workspaces. You can effortlessly glide between them using three-finger swipes.

* **Open the Overview**: Swipe **UP** with **three fingers**.
  * This reveals all your open windows and your current workspaces.
* **Open the App Grid**: While in the Overview, swipe **UP** again with **three fingers**.
  * This opens the application launcher showing all installed apps.
* **Close Overview / App Grid**: Swipe **DOWN** with **three fingers**.
  * This returns you to your active desktop window.
* **Switch Workspaces**: Swipe **LEFT** or **RIGHT** with **three fingers**.
  * This allows you to instantly switch between your virtual desktops. The animation exactly follows the movement of your fingers.

## Browsing and Scrolling

* **Scroll**: Swipe **UP/DOWN/LEFT/RIGHT** with **two fingers** to scroll through web pages, documents, and lists.
* **Zoom**: **Pinch** to zoom in, and **spread** to zoom out with **two fingers**. This is incredibly smooth in Wayland-native applications like the web browser or image viewer.
* **Right Click**: Tap with **two fingers** to simulate a right-click.

!!! tip "Natural Scrolling"
    By default, AnduinOS uses "Natural Scrolling" (moving your fingers up pushes the content up). If you prefer traditional scrolling, you can turn this off in **Settings** -> **Mouse & Touchpad** -> Disable **Natural Scrolling**.
