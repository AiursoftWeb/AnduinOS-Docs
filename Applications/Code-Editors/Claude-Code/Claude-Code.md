# Claude Code

!!! tip "AnduinOS Verified App - Proprietary"

    [Claude Code](https://claude.ai/code) is an AnduinOS verified app and it runs awesome on AnduinOS.

Claude Code is an agentic command-line interface (CLI) tool designed by Anthropic for software development. It runs directly in the terminal to read, write, edit, and test code across entire projects. It acts as an autonomous pairing partner, allowing developers to debug, refactor, and commit changes using natural language commands.

To install Claude Code on AnduinOS, you can run:

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

That's it! You now have Claude Code installed on your AnduinOS system. You can start using it by running the `claude` command in your terminal.

!!! warning "Unable to automatically upgrade this application"

    The above command only installs the application. If you run `sudo apt upgrade`, it won't upgrade it automatically. You will need to manually rerun the above command to upgrade.

    This is because the software provider didn't setup a repository for automatic updates. You will need to check the official website for updates.
